# 🏥 HealthAI Platform: Enterprise Medical Intelligence

An advanced, full-stack AI healthcare analytics system designed for disease prediction, clinical symptom analysis, and automated medical report interpretation.

![HealthAI Preview](https://via.placeholder.com/1200x600/0d1117/0ea5e9?text=HealthAI+Platform)

## 🚀 Key Features

*   **Multi-Disease Prediction**: Real-time ML models for Diabetes, Heart Disease, and Liver Disease.
*   **NLP Symptom Analysis**: Intelligent matching of 100+ diseases from natural language patient descriptions.
*   **OCR Medical Scanner**: Automated extraction and analysis of clinical lab reports (PDF/JPG).
*   **Interactive Analytics**: Vibrant, dynamic visualizations using Recharts and Framer Motion.
*   **Medical Knowledge Base**: A searchable clinical dictionary of symptoms, specialists, and precautions.

## 🛠️ Technology Stack

*   **Frontend**: React 19, Vite, Tailwind CSS v4, Lucide Icons, Recharts.
*   **Backend**: FastAPI (Python), SQLAlchemy, SQLite, Pydantic v2.
*   **AI/ML**: Scikit-Learn, Joblib, NLTK, NumPy, Pandas.
*   **Image Processing**: Tesseract OCR, OpenCV.

## 📦 Getting Started

### 1. Prerequisites
*   Node.js (v18+)
*   Python (3.9+)
*   Tesseract OCR (for report scanning)

### 2. Backend Setup
```bash
cd backend
pip install -r requirements.txt
python run.py
```
*API will be available at http://localhost:8000*

### 3. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
*Frontend will be available at http://localhost:5173*
